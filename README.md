# Airline Delay Analytics

A Python-based case study to analyze airline delays using historical flight data.

## Features
- Load and clean airline delay data from CSV files  
- Calculate average delays by airline and airport  
- Analyze delay trends and distributions  
- Visualize data using charts and heatmaps  
- Filter data by airline, airport, or month  
- Export statistics and plots  

## Tech Stack
Python, Pandas, Matplotlib, CSV Handling, OOP

## Purpose
Academic case study demonstrating data analysis and visualization in Python.
